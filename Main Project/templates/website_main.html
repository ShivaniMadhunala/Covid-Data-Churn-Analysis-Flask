Health Insurance Churn Analysis and Risk Classifier

About the Project

Our Project mainly focuses on Health Insurance companies. As a company grows it will automatically face some problems. Here we solved top two problems i.e., Churn Analysis and Risk Classification. This project is an intelligent solution using ML for the insurance domain. It uses classification for performing churn and risk analysis on the customers based on their demographics. We then use Flask to vizualize the data and display these results on a webpage.

Machine Learning Implentation

The following things hve been used to find solutions for these problems

  1. Artificial Neural Networks
  2. Flask
  3. Tensorflow
  4. SMOTE - Oversampling
